Getting Started

## Prerequisites

Install Ionic
```
npm install -g cordova ionic
```
Install dependencies
```
npm install
```
## Run the app
```
ionic serve --lab
```
## Build
iOS :
```
cordova platform add ios
```
Android :
```
cordova platform add android
```
## Built With
Ionic - Hybrid mobile app framework
